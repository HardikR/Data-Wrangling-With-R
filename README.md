# Data-Wrangling-1
Practical examples of data wrangling in R, covering the complete process from raw data cleaning to tidy, analysis-ready datasets. Includes use of tidyverse packages for real-world datasets and reproducible workflows.

- Utilising leading open-source software, R, to address and resolve data wrangling tasks.
- Selecting, performing, and justifying data validation processes for raw datasets to satisfy quality requirements.
- Understanding data structures and variable types.
